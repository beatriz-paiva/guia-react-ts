# Criando um projeto React com Vite

## Crie o projeto

Entre na pasta onde deseja criar seus projetos:

```bash
mkdir Projetos
cd Projetos
```

Crie o projeto:

```bash
npm create vite@latest
```

---

## Escolha as opções

Quando o assistente do Vite aparecer:

```text
Project name: meu-projeto

Framework:
> React

Variant:
> TypeScript
```

---

## Entre no projeto

```bash
cd meu-projeto
```

Instale as dependências:

```bash
npm install
```

---

## Execute o projeto

```bash
npm run dev
```

Você verá algo semelhante a:

```text
VITE v7.x.x ready in xxx ms

➜ Local: http://localhost:5173/
```

Abra o endereço informado no navegador.
