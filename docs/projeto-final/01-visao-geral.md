# Visão Geral do Projeto

Este projeto final integra **todos os conceitos** do guia em uma aplicação real: o **portal-admin** (Admin Apolo).

## Funcionalidades

- Login com autenticação JWT
- Sidebar colapsável com navegação
- Dashboard com cards de resumo
- CRUD completo de usuários
- Busca, paginação e filtros
- Tema claro/escuro

## Stack

| Tecnologia | Uso |
|---|---|
| React 19 + TypeScript | UI e tipagem |
| Vite 8 | Build |
| Tailwind v4 | Estilização |
| React Router v7 | Navegação |
| TanStack Query | Dados do servidor |
| zustand | Estado de UI |
| axios | HTTP |
| react-hook-form + Zod | Formulários |
